Build Python applications the Node.js way

upython is Python-loader engine that behaves similar to Node.js. It provides a require() that is used to load other upython packages. Packages can be installed with upm.

Register to upload Packages

You have to register an account using the upm register command on the command-line. Afterwards, you will receive a link to validate your e-mail address. From that point, you can upload packages to the registry. 

$ uom dist
$ upm upload dist/mypackage-1.0.0.tar.gz
